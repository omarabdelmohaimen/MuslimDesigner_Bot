# Quran Media Bot Pro (Reply Keyboard)

بوت تيليجرام لإدارة:
- كرومات
- تصاميم
- مناظر طبيعية

## المزايا
- الصفحة الرئيسية بـ Reply Keyboard مثل الشكل الذي طلبته
- قوائم السور تلقائيًا
- لوحة تحكم للأدمن بالأزرار فقط
- إضافة محتوى من داخل البوت
- حذف عنصر واحد أو حذف الكل
- الإحصائيات
- تخزين في JSON

## التشغيل
1. انسخ `.env.example` إلى `.env`
2. ضع `BOT_TOKEN` و `ADMIN_ID`
3. ثبّت المتطلبات:

```bash
pip install -r requirements.txt
```

4. شغّل البوت:

```bash
python bot.py
```

## ملاحظات
- ملفات الصور/الفيديو/الملفات تُحفظ كـ `file_id` داخل JSON.
- استخدم `/admin` لفتح لوحة التحكم.
- لو أردت إضافة شيخ جديد ضمن القوائم، أضفه إلى `data/content.json` داخل `settings.default_sheikhs`.
