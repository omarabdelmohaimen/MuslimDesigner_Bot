# Quran Media Bot Pro

بوت تيليجرام لإدارة:
- كرومات
- تصاميم
- مناظر طبيعية

## المزايا
- الصفحة الرئيسية بالأزرار
- قوائم السور تلقائيًا
- لوحة تحكم للأدمن
- إضافة محتوى من داخل البوت
- حذف عنصر واحد أو حذف الكل
- إحصائيات
- تخزين في JSON

## التشغيل
1. انسخ `.env.example` إلى `.env`
2. ضع `BOT_TOKEN` و `ADMIN_ID`
3. ثبّت المتطلبات:

```bash
pip install -r requirements.txt
```

4. شغّل البوت:

```bash
python bot.py
```

## ملاحظات
- إضافة المحتوى تتم من داخل لوحة الأدمن عبر الأزرار.
- ملفات الصوت/الفيديو/الصور تُحفظ كـ `file_id` داخل JSON.
- لو أردت إضافة شيخ جديد ضمن لوحة الأزرار، أضفه إلى `data/content.json` داخل `settings.default_sheikhs`.
