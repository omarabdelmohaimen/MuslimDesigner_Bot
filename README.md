# Quran Media Bot Pro (Reply Keyboard)

بوت تيليجرام لإدارة:
- كرومات
- تصاميم
- مناظر طبيعية

## المزايا
- الصفحة الرئيسية بـ Reply Keyboard
- قوائم السور تلقائيًا
- بحث عن السور داخل القسم
- لوحة تحكم للأدمن بالأزرار فقط
- إضافة محتوى من داخل البوت
- إضافة أسماء شيوخ جديدة من الأدمن
- حذف عنصر واحد أو حذف الكل
- الإحصائيات
- حفظ البيانات في Supabase مع نسخة محلية احتياطية

## التشغيل
1. انسخ `.env.example` إلى `.env`
2. ضع `BOT_TOKEN` و `ADMIN_ID`
3. أنشئ مشروع Supabase وضع:
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_ROLE_KEY`
4. ثبّت المتطلبات:

```bash
pip install -r requirements.txt
```

5. شغّل البوت:

```bash
python bot.py
```

## إعداد قاعدة البيانات في Supabase
شغّل الملف `supabase_setup.sql` داخل SQL Editor في Supabase.

## ملاحظات
- البوت يخزن كامل محتوى الكرومات/التصاميم/المناظر الطبيعية داخل صف JSONB واحد في Supabase، مع حفظ نسخة محلية كنسخة احتياطية.
- ملفات الصور أو الفيديو أو الملفات تُحفظ كـ `file_id` داخل البيانات.
- استخدم `/admin` لفتح لوحة التحكم.
- داخل قسم السور يمكنك استخدام زر `بحث عن سورة` للعثور على السورة بسرعة.
- من لوحة التحكم يمكنك استخدام `إضافة شيخ جديد` لإضافة أسماء شيوخ جديدة إلى القوائم.
